JavaScript

Origem

O JavaScript foi criado em 1995 por Brendan Eich, enquanto ele trabalhava na Netscape, com o objetivo de adicionar dinamismo às páginas web. Originalmente chamado de Mocha e depois LiveScript, foi rebatizado como JavaScript para aproveitar a popularidade do Java na época, embora as duas linguagens sejam muito diferentes. Seu principal propósito era ser uma linguagem de script executada no navegador, permitindo a interação direta com o HTML e CSS para criar páginas web dinâmicas e interativas. A linguagem rapidamente se popularizou, sendo adotada por outros navegadores e, em 1997, foi padronizada pela ECMA International como ECMAScript.

Conceito

  • Linguagem de programação interpretada: JavaScript é executado diretamente pelo navegador sem a necessidade de compilação.
  • Orientada a eventos: Permite que scripts respondam a eventos de interação do usuário, como cliques e teclas pressionadas.
  • Baseada em protótipos: A orientação a objetos no JavaScript é feita através de protótipos, e não classes, como em muitas outras linguagens
    • .
  • Multi-paradigma: Suporta programação orientada a objetos, funcional e imperativa.
  • Executado no lado do cliente: Originalmente, o JavaScript foi pensado para rodar no navegador, mas hoje, com o Node.js, também pode ser executado no lado do servidor.
